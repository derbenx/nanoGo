NanoGo Pro Guide

🚀 Getting Started

NanoGo Pro is a high-performance frontend for Google's Gemini models, supporting both Image Generation and AI Chat. To start, head to the Settings tab and enter at least one API Key.

🖼️ Image Generation (Create Tab)

  • Adding Images: Drag and drop image files anywhere on the app, or use the Add Images button. Use New Image to create a <GENERATE> placeholder for text-to-image tasks.
  • Creating Tasks: Select images using the checkboxes and click Add Task. You can merge multiple images into one task (e.g., for reference) by checking multiple boxes.
  • Source IDs: You can manually edit these. 1+2 will use both images. The field auto-filters non-existent IDs after 2 seconds.
  • Aspect Ratio: The app auto-picks the closest matching ratio when you add an image. Extreme ratios (1:4, 8:1) are exclusive to the Nano 2 tier.
  • Return Thought: Enable this to save the AI's reasoning as a .txt file alongside your image.
  • Execution:
    • Immediate: Runs tasks locally and sequentially. Best for single prompt-only tasks.
    • Batch: Submits tasks to Google's cloud for 50% cheaper processing. Recommended for large sets of images.

💬 AI Chat

A simple yet powerful multi-turn chat interface.

  • Personality: Set a system prompt to define how the AI behaves (e.g., "You are a coding expert").
  • Memory Slots: Controls how many previous message pairs are kept in context.
  • Initial Memory: If enabled, the very first exchange is always kept, helping the AI remember its primary goal throughout a long session.
  • Cost Estimate: Shows a character-based token cost approximation for the current input plus active memory.

⚙️ API & Settings

  • Dual Keys: Use separate Paid and Free keys. Toggle Free Mode in the Chat or Create tabs to switch between them instantly.
  • Batch Monitoring: The app persists Job IDs in jobs.txt. It will resume tracking them even if you restart the application.
  • Resets: Use the Reset buttons next to individual sections to restore default values without wiping your API keys or paths.

⌨️ Keyboard & UI Shortcuts

  • Tab Cycling: Use Ctrl + Tab and Ctrl + Shift + Tab to quickly switch views.
  • Context Menus: Right-click images or tasks for advanced options (Delete, Duplicate, Show Image, Change Reference).
  • Layout: Drag the vertical bar in the Create tab to resize your workspace. This position is saved automatically.
  • Sessions: Use Save/Load Session to keep track of complex task lists. You can also drop a .json session file directly onto the app.
